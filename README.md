Movie Master Pro 

Movie Master Pro is a modern, interactive web application where users can browse movies, add their favorites to a watchlist, and rate or review them. The app offers a smooth and enjoyable user experience with advanced filtering and theme toggling options.

Itâ€™s built using React, styled with Tailwind CSS and DaisyUI, powered by Node.js / Express for the backend, and uses MongoDB to store data.

 Features

Movie Browsing: Explore popular and trending movies in one place.
Multi-Genre Filter: Filter movies by multiple genres at once.
Rating Range Filter: Filter movies based on your preferred rating range.
Watchlist: Save your favorite movies to a personal watchlist for later viewing.
Reviews & Ratings: Users can rate movies and leave reviews.
Theme Toggle: Switch between light and dark modes for a personalized experience.
User Authentication: Secure sign-up and login functionality via Firebase.
