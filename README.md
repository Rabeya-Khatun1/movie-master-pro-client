## ‚ú® Project Overview
MovieMaster Pro allows users to:  
- Browse all movies with detailed information  
- Add, edit, or delete movies (CRUD)  
- View personal movie collections and watchlist  
- Filter movies by genre, rating, and release year  
- Search movies using dynamic search functionality  
- Switch between light and dark themes

## üîë Core Features

### üè† Home Page
- Hero section with **Featured Movies Carousel**  
- Statistics: Total Movies, Total Users  
- Top Rated Movies: Display top 5 highest-rated movies  
- Recently Added Movies: Show latest 6 movies  
- Genre Section: Static list of genres (Action, Drama, Comedy, etc.)  
- About Platform Section: Overview of MovieMaster Pro  
- Smooth animations for better UX  

### üì± Authentication System
- **Login Page:** Email/Password & Google Sign-In  
- **Register Page:** Name, Email, Photo URL, Password validation  
- Success/Error handling via **toast notifications**  

### üé¨ Movie Management (CRUD)
- **All Movies Page:** Poster, Title, Genre, Rating, Release Year  
- **Movie Details Page:** Full movie info, Edit/Delete for owners  
- **Add Movie Page:** Form input for new movies  
- **Update Movie Page:** Pre-filled editable form (except Added By)  
- **Delete:** Confirmation modal + UI update  
- **My Collection Page:** User-specific movies  
- **Watchlist Feature:** Add favorite movies to watchlist  

### ‚ö° Advanced Features
- **Protected Routes:** Only logged-in users can add/update movies  
- **Advanced Filtering:** By multiple genres or rating range  
- **Theme Toggle:** Light ‚Üî Dark mode  
- **Loading States & Error Handling:** Spinners and custom 404 pages
